## 🎓 Best Practices Summary

✅ **DO**:
- Use two-tier caching (SharedPreferences + SQLite)
- Increment database version for each change
- Add `cached_at` timestamp to all tables
- Implement background updates
- Clear cache after mutations
- Test offline functionality

❌ **DON'T**:
- Cache authentication tokens in SQLite
- Cache without expiration strategy
- Forget migration logic
- Skip error handling
- Cache filter-heavy lists completely
